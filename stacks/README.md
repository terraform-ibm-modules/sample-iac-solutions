# Terraform Stacks Example

An example that shows how to deploy a managed PostgreSQL database on IBM Cloud using Terraform Stacks.

The following resources are provisioned:

- A Resource Group for organizing and managing cloud resources.
- An IBM Cloud Databases for PostgreSQL instance with automatic backups, high availability, and built-in security.

## Deployment

For more information on deployment steps, see [Deploy Terraform IBM Modules using HCP Terraform Stacks](https://cloud.ibm.com/docs/ibm-cloud-provider-for-terraform?topic=ibm-cloud-provider-for-terraform-deploy-tim-using-hcp-stacks).
